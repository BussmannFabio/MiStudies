## FAQ

| Pergunta | Resposta atual | Status |
|---|---|---|
| Preciso assinar o Hub para ter aulas publicadas? | Sim. O Hub é a assinatura que mantém sua página ativa e as aulas acessíveis aos alunos. Sem o Hub, as aulas ficam criadas mas não publicadas. Você pode começar com o Hub Starter (R$ 49/mês) e fazer upgrade conforme crescer o volume de conteúdo. | |
| Posso misturar tiers diferentes numa mesma assinatura? | Sim. Cada aula é cobrada individualmente no tier escolhido para aquela aula. Você pode ter 3 aulas Essencial, 2 Pro e 1 Full Studio no mesmo mês — sem nenhuma restrição de mix. | |
| O que acontece se eu cancelar o Hub? | As aulas ficam arquivadas no nosso sistema. Ao reativar a assinatura, tudo volta ao ar imediatamente. Não há perda de conteúdo já produzido. | |
| Quanto tempo leva para receber uma aula depois de enviar o PDF? | Essencial: 3–5 dias úteis. Pro: 5–7 dias úteis. Full Studio: 7–10 dias úteis. Prazos contados a partir da confirmação do pagamento e recebimento do material completo. | |
| O onboarding é cobrado uma única vez? | Sim. A taxa de onboarding é paga uma única vez na entrada. Depois disso, você paga apenas pelas aulas criadas e pela assinatura mensal do Hub. | |
| Posso fazer upgrade do meu plano de onboarding depois? | Sim. Caso queira evoluir do Básico para o Branded ou do Branded para o Institucional, cobramos apenas a diferença de valor entre os planos. | |
